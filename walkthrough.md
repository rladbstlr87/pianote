# Pianote Final Implementation Walkthrough

이 프로젝트는 피아노 음원 변환의 정확도를 극한으로 높이고, 사용자 편의성을 극대화한 최종 단계에 도달했습니다.

## 🚀 업그레이드 하이라이트

1. **SOTA (State-of-the-Art) 엔진 탑재**
2. **럭셔리 웹 인터페이스 (Browser-based UI)**

## 🛠️ 기술 스택
- **Engine**: ByteDance High-res Piano Transcription
- **Backend**: Flask
- **Frontend**: Vanilla HTML5, CSS3, JS

## 📖 사용 방법
- `python app.py` 실행 후 `localhost:5000` 접속
- 음원 업로드 시 즉시 MIDI 변환 및 다운로드 가능
